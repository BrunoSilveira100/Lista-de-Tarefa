# Lista de Tarefas — Projeto

Pequena aplicação de lista de tarefas (HTML/CSS/JS puro).

Como usar

- Abra o arquivo `Tarefas.html` no navegador (duplo clique ou `File → Open`).
- Digite uma tarefa no campo e clique em **+ Adicionar** ou pressione `Enter`.
- Clique no círculo à esquerda ou no próprio item para marcar como concluída (riscar).
- Clique no botão `✕` no final do item para remover.
- Use os filtros (Todas / Ativas / Concluídas) para visualizar o conjunto desejado.
- As tarefas são salvas automaticamente no `localStorage` do navegador e persistem após recarregar.

Validações

- Não é permitido adicionar tarefas vazias; uma mensagem de erro será exibida.

Arquivos importantes

- `Tarefas.html` — arquivo principal com HTML, CSS e JavaScript.
